PSV破解教程3.60~3.68

发表于 2020-02-26 | 更新于 2020-03-03
吃灰的PSV也能玩出新花样

破解

参考了好几个网站和文档，首先说我自己的破解方式。我是参考吾爱破解网的一个帖子：https://www.52pojie.cn/thread-938461-1-1.html 破解的PSV是2000型号，系统是3.68所以按照教程来就可以了。

  • 第一步：首先下载Qcma并安装，网址：https://codestation.github.io/qcma/选择windows linstaller（0.4.1）下载并安装。一切按默认设置即可（注：不要修改任何设置）

  • 第二步：下载auto.h-encore（https://pan.baidu.com/s/1ZzGP4UbuEH8fRDTjy8oJUw 提取码: bh66）

这个时候重启一下电脑，psv一定要开飞行模式并将wifi连接关闭，若提示更新系统请重启psv再进行尝试（不要更新）

  • 第三步：打开Qcma 并将psv以usb形式连接到电脑并在psv上点击
  • 第四步：打开auto.h-encore，将【删减bitter smile】那个选项关闭，点击启动。这一步软件将会下载很多额外的程序包，包括 psvimgtools pkg2zip 等，一般会从github上下载，速度很慢，而且由于这个程序没有设计下载进度条提示，请务必耐心等待（挂梯子会快点）。

下载完成之后会弹出【提示框】，选择USB然后跟着程序提示安装就可以。这里程序让你使用PSV打开【文件管理】，将【h-encore】拷贝到PSV上的操作。

拷贝完成后，将PSV的飞行模式关闭，WIFI打开连上网络。

  • 第五步：完成以上四步后会发现psv多出一个【h-encore】的程序，点击进入黑色界面。

第二项和第三项分别点击O进行安装，然后选择第一项退出。这个时候桌面多出一个【文件管理】的图标。

  • 第六步：进入设置，菜单当中多了一个【HENkaku】的选项，点进去将【启用不安全自制软件打勾✔】

破解完成。

VPK游戏安装

游戏下载可以到http://www.vpkgame.com/ 下载VPK文件

  • 第一步：将PSV通过USB连接笔记本，打开QCMA找到音乐安装的文件路径，windows系统一般来说都是系统音乐文件夹。

  • 第二步：将下载的vpk文件改名为mp3文件

  • 第三步：在PSV上将PC中的【音乐】打开，选择刚刚改名的MP3文件，拷贝到PSV上

  • 第四步：打开【文件管理系统】，找到music的文件夹，将MP3后缀修改为vpk，点击安装。（music文件夹的位置在容量最大的那个文件夹里）

游戏安装完成

MAI游戏安装

由于直接安装VPK文件安装速度慢，而且容易安装失败。总结了MAI安装方法。

参考自贴吧：https://tieba.baidu.com/p/5778668494

mai工具链接：https://pan.baidu.com/s/10aInP8Gthuqk76hYwLJccg 密码：r3ff

  1. 首先下载mai工具然后安装【可以用VPK安装法】
  2. 然后打开vitashell，在根目录按start，将select设置为USB
  3. 将PSV连接到电脑，打开vitashell，按select，这个时候psv会以U盘形式连接，打开U盘。
  4. 将VPK文件改成ZIP进行解压，解压后放入一个英文数字的文件夹。【比如我下一个“0233_日中_PCSG00551_太鼓达人_0101”，解压之后改名就是PCSG00551】
  5. 将文件夹拷贝到U盘下【mai】文件夹里
  6. 启动mai工具，选择文件夹安装
    安装完成

存档问题

解决存档bug解决办法！

  1. 进去设置把时间自动更新去除！
  2. 把时间改成12小时制
  3. 把PSV\user\00\SAVEDATA_BACKUP文件夹删除！删除之前请备份好！在用电脑创建一个savedata_backup同名txt文件！不是文件夹！

记得去除后缀名！然后把这个文件放入之前删除的文件夹所在位置！搞定！目前有朋友测试成功！请大家自行测试！！！PS一下，对于FTP传savedata_backup同名txt文件，可以在PSV里找一个txt文件复制到PSV\user\00\下，改名SAVEDATA_BACKUP，这样就不用传txt文件了
